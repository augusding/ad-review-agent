8层检测维度详解

1. 文案违禁词

规则库 + DeepSeek 双层检测
  • 绝对化用语（最、第一、国家级）
  • 虚假宣传 / 夸大功效
  • 诱导性表述 / 伪造通知
  • 违禁金融承诺 / 医疗疗效
<500ms全品类

2. 图片内容安全

Qwen VL Max + 阿里云双重验证
  • 色情 / 低俗内容
  • 暴力 / 恐怖元素
  • 政治敏感 / 违禁标志
  • 虚假 UI 截图 / 伪系统通知
~5s全品类

3. 视频关键帧

5帧抽样 + Qwen VL Max
  • 关键帧画面合规检测
  • 与广告文案一致性
  • 品牌标识合规性
~15s (异步)视频素材

4. 视频音频

moviepy + DashScope ASR
  • 语音转写内容检测
  • 与画面/文案一致性
  • 违禁口播内容识别
~8s (异步)视频素材

5. 落地页一致性

httpx 抓取 + DeepSeek 比对
  • 广告承诺与落地页匹配
  • 价格 / 功能一致性
  • 落地页可访问性检测
  • 二次跳转 / 诱导下载
~3s全品类

6. 行业资质匹配

规则库 + 版号官方接口
  • 行业×资质要求映射
  • 游戏版号验证
  • 金融/医疗许可证核验
  • ICP 备案查询
<300ms金融/游戏/医疗

7. 平台专项规范

vivo 品牌保护 + 竞品检测
  • vivo 品牌保护规则
  • 竞品品牌提及检测
  • 素材尺寸/格式合规
  • 特殊位置投放规范
<200ms全品类

8. 跨维度一致性检测

Qwen VL 直判 + DeepSeek 语义比对
  • 文案 vs 图片：品牌冲突 / 品类错位
  • 文案 vs 落地页：价格承诺 / 功能兑现
  • 图片 vs 落地页：品牌标识一致性
  • 规则可配置热更新（阈值/正反例）
~3s图文混合素材

技术栈与模型选型

AI 模型层

DeepSeek Chat

文案语义分析 / 落地页比对
选型理由：中文理解强，成本低，API 稳定

Qwen VL Max

图片 / 视频视觉检测 / 图文一致性直判
选型理由：国内最强视觉模型，合规，支持多图输入，价格低

阿里云内容安全

违禁内容专项检测
选型理由：专项训练模型，准确率高，与 Qwen 双重验证

DashScope ASR

视频语音转写
选型理由：复用阿里云账号，识别准确率高
